Customer Churn Prediction 🤖
Overview
This machine learning project predicts whether a customer will churn based on age, monthly spending, and contract length.

Features
- Data preprocessing
- Train/test split
- Logistic Regression model
- Accuracy evaluation

 Technologies Used
- Python
- Pandas
- Scikit-learn

Why This Project?
Customer churn prediction is widely used in subscription businesses to reduce revenue loss and improve customer retention. This project demonstrates beginner-level ML workflow skills.
